# Background Remover

Backgound remover is a simple python program to remove the images background

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements.

```bash
pip -r install requirements.txt
```

## Usage

```bash
python app.py
```
